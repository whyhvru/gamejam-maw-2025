# MAW

Experimental UI-driven psychological horror game about madness, obsession and survival.

## About

Six months ago, he accepted a questionable offer — to rewrite a forbidden manuscript inside an abandoned monastery of the Initiated Order.

Seven entered. One returned.

Now he walks home to his family through a cursed forest known as the Maw.

Here, creatures thirst for blood, while the fog suffocates the flesh and denies death itself.

He no longer thinks about the manuscript.

Only about reaching home.

If that's even possible...

## Features

- Madness system where death increases insanity instead of ending the game
- Experimental interface-driven gameplay
- Narrative progression through inner monologues and narration
- Multiple action controls and reactive mechanics
- Black-and-white pixel-art aesthetic
- Atmospheric dark fantasy horror without jumpscares

## Controls

| Key | Action |
|---|---|
| Q | Cover ears |
| W | Walk forward (story progression) |
| E | Whistle |
| R | Hold breath |
| A | Strike with hand |
| S | Step back quickly |
| D | Drink sedative |
| F | Use flint |
| TAB | Check heartbeat |
| ENTER | Exit game |
